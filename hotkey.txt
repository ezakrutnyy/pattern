ctrl+J - сокращение всех команд в IDEA

psvm - создание метода  main со всей сигнатурой
sout - system.out.println
soutv - system.out.println("value"+value)
souf - system.out.printf("%s",s)
serr - System.err.println

*************Проходы по структурам*************************************************
I - создание цикла foreach (курсор на item)
iter - создание цикла foreach (курсор на итерируемом элемента)
fori - создание обычного цикла for с int-ключами
itar - проход по массиву
ritar - проход по массиву с конца
itco - проход по коллекции через итератор с for
itit - итератор с while
itli - итератор по листу , с индексами int

*************Условные операторы*************************************************
ifn - if args == null
inn - if args !=null

************Редактиррование кода*************************************************
shift+ -> или shift+ <- - выделение строчки
ctrl+shift+ стрелочка вверх или вниз - перенос выделеного кода вниз или вверх
ctrl+alt+t - выделенную строку обернуть в какую нибудь конструкцию if/for/while/exception
shift+end или shift+home - выделение строки с начала или с конца
ctrl+d - скопировать и перенести строку вниз
ctrl+y - удалить выбранную строку
ctrl+x - копирование в буфер и удаление строки
ctrl+w - пошаговое выделение элементов с исходного элемента
ctrl+shift+ w - пошаговое удаление выделенного дл исходного элемента
ctrl+ - раскрыть метод
ctrl -  скрыть метод
ctrl+[ или ctrl+] - перемещает курсор вверх по блокам или вниз
ctrl+alt+o - удаление всех неиспользуемых импортов
ctrl+g - переход на конкретную строку кода
ctrl+f - поиск текста
ctrl+N - поиск любого класса

**************Работа с визулизацией в редакторе*********
ctrl+тильда - выбор скина визуализации ide и стиля кода

**************Генерация классов*************************
alt+insert - создание конструкторов, геттеров и сеттеров и прочих методов
ctrl+alt+insert - создание классов, интерфейсов, файлов
ctrl+I - реализует абстрактный метод
***************Табуляция*********************************
TAB - табуляция кода в 4 пробела
TAB+Shift - убрать символ табуляции
ctrl+alt+i - отформатирует всю табуляцию в выделеннном коде

**************Комментарии**********************************
ctrl+/ - комментирует одну строчку и переносит курсор на новую строку
ctrl+shift+/ выделенный блок комментирует в многострочный комментарий /***/

**************Отображение необходимый панелей редактора********************
alt+1 - Project
alt+6 - TODO
alt+7 - структура данного класса и тд . все подписано на иконках
ctrl+alt+s - Общие настройки IDE
